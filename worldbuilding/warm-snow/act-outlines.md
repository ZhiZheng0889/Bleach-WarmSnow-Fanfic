# Act Outlines (stakes and faction links)

Act scaffolds tying clans/sects to objectives and captains.

- **Act I – Rukongai Breach**
  - Objective: Seed mimic relics; test Guard response.
  - Factions: Yue smugglers (tokens/lanterns), Xu “lost shipments,” Long conscripts as cover.
  - Captains/sects: Soi Fon/2nd patrols; Seven-Sword Shura aiding smugglers.
  - Stakes: If caught, Bai censors bury reports; if successful, Hellborne gains river corridors.

- **Act II – Seireitei Outer Wall**
  - Objective: Smuggle a Unique relic; install suppressors.
  - Factions: Bai layers seals, Long enforces banners, Yue ferries cargo.
  - Captains/sects: Byakuya encounters Blade Cage; Thunderfury cracks barriers.
  - Stakes: Bai gains leverage over nobles; Urahara/Mayuri counter-tech race begins.

- **Act III – Dangai/Frost Pockets**
  - Objective: Build a time-dilated forge; steal forbidden scrolls.
  - Factions: Bai frost wardens, Xu forge retinues, Shen tests sedatives on prisoners.
  - Captains/sects: Hitsugaya/Hinamori parallels; Bitter River Bodhi vs Bai ice; Soul Blade Shura skirmishes.
  - Stakes: If Bai keeps scrolls, Kido Corps gains monopoly; if stolen, Hellborne mimics get smarter.

- **Act IV – Hell Forge beneath Senzaikyu**
  - Objective: Mass-produce legendary mimics; bind Soul King scrap.
  - Factions: Xu ritualists, Chains Path escorts, Shen bio-fuel trials.
  - Captains/sects: Mayuri infiltrates; Chains Path vs 12th; Thunderfury/Seven-Sword disrupt forge arrays.
  - Stakes: Xu could arm clan militias with suppressor-proof arms; destruction weakens Hellborne logistics.

- **Act V – Royal Gambit**
  - Objective: Test Unique blades vs Royal Guard; steal Royal Key schema.
  - Factions: Bai spies, Long banners guarding gates, Yue/Shadow-Walker couriers.
  - Captains/sects: Squad Zero emissary duel; Paper Talismans mimic Bankai silhouettes.
  - Stakes: If success, Hellborne Paths leap in power; if failed, clans face purge and scapegoating.

- **Hueco Mundo DLC**
  - Objective: Weaponize mimics vs Hollows/Arrancar; harvest masks.
  - Factions: Shen poison trials, Yue Garganta ferries, Xu forging anti-Cero plating.
  - Captains/sects: Grimmjow/Arrancar clashes; Azure Dragon lightning; Wheels of Solar and Lunar stress-test dual elements.
  - Stakes: Shen toxins could breach masks; Bai wants Quincy notes; Urahara intervenes.
