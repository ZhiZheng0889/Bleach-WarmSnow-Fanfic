# Matchups: Hellborne Path Heads vs Captains

Quick pairing suggestions for duels and set-piece fights. Swap captains as you like to fit arcs.

- **Blade Path (Asahi/Murasame)** ↔ **Kenpachi / Byakuya**
  - Blade vs raw power; Murasame’s silence vs Zanpakuto spirits; Kenpachi brute-forces Blood Calyx rage boosts.

- **Frost Path (Rime/Haku)** ↔ **Hitsugaya / Hinamori**
  - Frost zones vs ice control; who owns the freeze; Rime tries to stack Senkaimon-like traps.

- **Thunder Path (Raiden/Noa)** ↔ **Soifon / Kyoraku**
  - Thunder cracks Kido nets; Soifon’s speed answers dashes; Kyoraku’s games disrupt lightning rhythms.

- **Poison Path (Venia/Sango)** ↔ **Unohana / Mayuri**
  - Medical mind versus poisons and counter-poisons; Sango flips toxins, Unohana counters with healing pressure; Mayuri wants their data.

- **Blood Path (Okuro/Shion)** ↔ **Kenpachi / Komamura**
  - Blood rage engines vs tanky frontline; Shion siphons Reiatsu to heal squads; Komamura’s armor resists initial spikes.

- **Flying Swords Path (Yatori/Setsuna)** ↔ **Byakuya / Ukitake**
  - Sword swarms vs Senbonzakura petals; trajectory bending via Tideway Tokens; Ukitake redirects spirit projectiles.

- **Chains Path (Guren/Haru)** ↔ **Soi Fon / Renji**
  - Snatch-and-bind vs Shunko and Zabimaru reach; suppressor links to delay releases; Chains yank mid-flash step.

- **Furnace Path (Iori/Riku)** ↔ **Kurotsuchi / Urahara**
  - Mobile forge rerolls vs science captains; overheat enemy weapons; Mayuri/Urahara counter-engineer mid-fight.

- **Paper Talismans Path (Suzu/Mei)** ↔ **Kyoketsu/Kido Corps / Kyoraku**
  - Sealcraft and false signatures vs Kido specialists; Kyoraku’s games punish predictable talisman timing.

- **Specters Path (Kageho/Yomi)** ↔ **Shinji / Aizen (past) / Yoruichi**
  - False spirits and afterimages vs inverted senses; Shinji reverses orientation; Yoruichi hunts the caster behind wraiths.
