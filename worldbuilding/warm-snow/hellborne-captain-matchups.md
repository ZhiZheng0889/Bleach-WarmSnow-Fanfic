# Matchups: Hellborne Path Heads vs Captains

Quick pairing suggestions for duels and set-piece fights. Swap captains as you like.

- **Blade Path (Zhao Lie / Mu Ren)** ? **Kenpachi / Byakuya**
  - Blade vs raw power; Mu Ren’s silence vs Zanpakuto spirits; Kenpachi brute-forces Blood Calyx rage boosts.

- **Frost Path (Leng Xue / Han Bing)** ? **Hitsugaya / Hinamori**
  - Frost zones vs ice control; who owns the freeze; Leng Xue stacks Senkaimon-like traps.

- **Thunder Path (Lei Guang / Dian Yu)** ? **Soifon / Kyoraku**
  - Thunder cracks Kido nets; Soifon’s speed answers dashes; Kyoraku’s games disrupt lightning rhythms.

- **Poison Path (Du Ming / Yao Zhi)** ? **Unohana / Mayuri**
  - Medical mind versus poisons and counter-poisons; Yao Zhi flips toxins, Unohana counters with healing pressure; Mayuri wants their data.

- **Blood Path (Okuro Gen / Shion Akeboshi)** ? **Kenpachi / Komamura**
  - Blood rage engines vs tanky frontline; Shion siphons Reiatsu to heal squads; Komamura’s armor resists initial spikes.

- **Flying Swords Path (Jian Fei / Ling Jian)** ? **Byakuya / Ukitake**
  - Sword swarms vs Senbonzakura petals; trajectory bending via Tideway Tokens; Ukitake redirects spirit projectiles.

- **Chains Path (Guren Fudo / Haru Natsume)** ? **Soi Fon / Renji**
  - Snatch-and-bind vs Shunko and Zabimaru reach; suppressor links to delay releases; Chains yank mid-flash step.

- **Furnace Path (Rong Yan / Huo Xin)** ? **Kurotsuchi / Urahara**
  - Mobile forge rerolls vs science captains; overheat enemy weapons; Mayuri/Urahara counter-engineer mid-fight.

- **Paper Talismans Path (Fu Shen / Zhi Wen)** ? **Kyoketsu/Kido Corps / Kyoraku**
  - Sealcraft and false signatures vs Kido specialists; Kyoraku’s games punish predictable talisman timing.

- **Specters Path (You Hun / Ling Ying)** ? **Shinji / Aizen (past) / Yoruichi**
  - False spirits and afterimages vs inverted senses; Shinji reverses orientation; Yoruichi hunts the caster behind wraiths.
